# upgreyd - Advanced Fitting Algorithm Improvements

This directory contains a significant "upgrade" to the fitting algorithms, documented through a conversation with Gemini (AI).

## Contents

- **full_reason.md**: A detailed log of the conversation and reasoning behind the improvements. It explains the transition from heuristic calibration to a more rigorous model-based approach.
- **new_calibrations.ino**: The resulting Arduino code incorporating the improved algorithms.
- **fitter.cpp** & **fitter_claude.cpp**: C++ implementations (likely for testing or sub-modules) of the fitting logic, including versions generated by Claude.

## Key Upgrades

- **Model-Based Rint Calibration**: Instead of simple heuristics, the system uses a circuit model with `Voc` and `Rsrc` in series with a load `Rload`.
- **Gradient Descent for Resistance**: Implements Gradient Descent to iteratively refine estimates of both internal source resistance (`Rsrc`) and load resistance (`Rload`) by minimizing the error between predicted and measured voltages at different duty cycles.
- **Dual-Point Measurement**: Calibration now uses measurements at two different duty cycles (Low and High) to better triangulate the resistance values.
- **Partial Derivatives**: The code uses derived partial derivatives of the voltage equations to calculate the gradients for the optimization process.
